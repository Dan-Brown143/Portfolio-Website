# Personal Portfolio Website

A modern, responsive portfolio website made with HTML, Css, and JavaScript. Perfect for showcasing projects and skills to potntial employers.

## Features

- Clean design with smooth animations
- Works on all devices
- Professional colour scheme and typography
- Sections for About, Projects, Skills and Contact
- Smooth scroll navigation


## Setup Instructions for VS Code

### 1. Install VS Code from: https://code.visualstudio.com/

### 2. Install Recommended Extensions

Open VS Code and install:

- **Live Server** - Launch a local server that reloads live
- **HTML CSS Support** - Intellisense for HTML/CSS

To install:
1. Click the extensions icon on the left sidebar, or press **Ctrl+Shift+X**
2. Search for each extension
3. Click Install

### 3. Launch the Website

**Using the live server extension**
1. In the bottom right click **Go Live**
2. The browser will automatically open to **http://localhost:5500**
3. Any changes you make and save will automatically reload on the website

## Troubleshooting

**Styles not loading**
- Check that **styles.css** is in the same folder as **index.html**
- Check the file name spelling matches the tags

**JavaScript not working**
- Check the F12 console for errors
- Ensure **script.js** is in the same folder as **index.html**
- Check the file name matches the tags

**Live server not working**
- Make sure the extension is installed
- Check if another server is running on port 5500
- Restart your VS Code
